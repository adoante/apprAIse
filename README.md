## Set up

### First create a virtual environment

```bash
python -m venv .venv
```

### Activate environment (Windows Powershell)

```bash
.\.venv\Scripts\Activate.ps1
```

### Install requirements

```bash
pip install -r requirements
```

## Run app

```bash
fastapi dev .\api\run_api.py
```
